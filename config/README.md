# ملفات الإعدادات

هذا المجلد يحتوي على ملفات الإعدادات الثابتة.

## الملفات

- `languages.json` - قائمة اللغات المدعومة
- `translation-config.json` - إعدادات محركات الترجمة
- `stt-config.json` - إعدادات محركات STT
- `expert-system.json` - إعدادات نظام الخبراء
- `ui-config.json` - إعدادات واجهة المستخدم
- `performance.json` - إعدادات الأداء
- `security.json` - إعدادات الأمان

## المسؤوليات

- تخزين الإعدادات الثابتة
- إدارة معرفات اللغات
- تكوين الخدمات الخارجية
- تحديد حدود الاستخدام
- تحديد قواعد العمل

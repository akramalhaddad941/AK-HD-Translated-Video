# تعريفات TypeScript

هذا المجلد يحتوي على جميع تعريفات الأنواع والواجهات.

## الملفات

- `index.ts` - ملف فهرس للتصدير
- `chrome-extension.ts` - تعريفات Chrome Extension
- `speech.ts` - تعريفات Speech-to-Text
- `translation.ts` - تعريفات Translation
- `audio.ts` - تعريفات Audio Processing
- `ui.ts` - تعريفات واجهة المستخدم
- `config.ts` - تعريفات الإعدادات
- `expert-system.ts` - تعريفات نظام الخبراء
- `video.ts` - تعريفات الفيديو
- `events.ts` - تعريفات الأحداث

## المسؤوليات

- ضمان type safety
- توثيق واجهات البرمجة
- تقليل أخطاء التطوير
- تحسين تجربة التطوير
- تسهيل الصيانة والتطوير

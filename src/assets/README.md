# الموارد والصور

هذا المجلد يحتوي على جميع الموارد الثابتة مثل الصور والأيقونات.

## الملفات

### الأيقونات
- `icons/` - أيقونات الإضافة
  - `icon16.png`
  - `icon32.png`
  - `icon48.png`
  - `icon128.png`

### الصور
- `images/` - صور المستخدم
  - `flags/` - أعلام اللغات
  - `ui/` - صور واجهة المستخدم
  - `backgrounds/` - خلفيات

### الخطوط
- `fonts/` - ملفات الخطوط

### الصوت
- `sounds/` - أصوات التنبيهات

## المسؤوليات

- تخزين الموارد الثابتة
- تحسين تحميل الصور
- ضمان التوافق مع المتصفحات
- إدارة إصدارات الموارد
- ضغط وتحسين الملفات

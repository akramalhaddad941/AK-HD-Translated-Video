# Content Scripts - فحص الفيديو

هذا المجلد يحتوي على Content Scripts التي تعمل في صفحات الويب.

## الملفات

- `main.ts` - الملف الرئيسي للـ Content Script
- `video-detector.ts` - كاشف عناصر الفيديو
- `overlay-manager.ts` - مدير التراكب على الفيديو
- `event-listener.ts` - مستمع الأحداث
- `floating-icons.ts` - الأيقونات العائمة

## الواجبات

- كشف عناصر الفيديو في الصفحات
- إدراج واجهة المستخدم على الفيديو
- معالجة أحداث تشغيل الفيديو
- التواصل مع Service Worker
- عرض الترجمات والتفاعل معها

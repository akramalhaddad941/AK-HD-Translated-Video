# دوال مساعدة

هذا المجلد يحتوي على الدوال والأدوات المساعدة.

## الملفات

- `index.ts` - ملف فهرس للتصدير
- `utils.ts` - دوال عامة
- `storage.ts` - وظائف التخزين
- `language.ts` - دوال اللغات
- `time.ts` - دوال الوقت
- `network.ts` - دوال الشبكة
- `validation.ts` - دوال التحقق
- `formatting.ts` - تنسيق البيانات
- `debounce.ts` - دوال التأخير
- `logger.ts` - دوال السجلات

## المسؤوليات

- توفير دوال مساعدة عامة
- تبسيط العمليات المعقدة
- تحسين كفاءة الكود
- ضمان إعادة الاستخدام
- تحسين الأداء

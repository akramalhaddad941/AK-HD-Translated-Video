# النواة الأساسية للنظام

هذا المجلد يحتوي على النواة المنطقية والهيكلية للنظام.

## الملفات

- `engine.ts` - محرك النظام الأساسي
- `config-manager.ts` - مدير الإعدادات
- `expert-system.ts` - نظام الخبراء الذكي
- `event-dispatcher.ts` - موزع الأحداث
- `logger.ts` - نظام السجلات

## المسؤوليات

- التنسيق بين جميع المكونات
- اتخاذ القرارات الذكية
- إدارة تدفق البيانات
- مراقبة الأداء
- تطبيق قواعد العمل

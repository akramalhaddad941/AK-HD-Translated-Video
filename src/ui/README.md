# واجهة المستخدم

هذا المجلد يحتوي على جميع مكونات واجهة المستخدم.

## الملفات

- `popup.tsx` - النافذة المنبثقة
- `options.tsx` - صفحة الإعدادات
- `components/` - مكونات React القابلة للإعادة
- `contexts/` - React Contexts
- `hooks/` - React Hooks المخصصة
- `locales/` - ملفات الترجمة

## التقنيات

- React 18
- Material-UI 5
- React-i18next
- TypeScript
- Emotion (CSS-in-JS)
